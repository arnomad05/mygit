# mygit
Проект
Для сборки проекта выполните:
    make

Для очистки:
    make clean

Для пересборки:
    make rebuild

Реализованные команды

  init          - инициализация репозитория
  add <file>    - добавление файла в staging
  remove <file> - удаление файла из staging
  commit <msg>  - создание коммита
  log [-n N]    - просмотр истории коммитов
  diff <commit> - сравнение коммитов
  status        - состояние репозитория
  checkout <ref>- переключение веток
  branch [name] - создание/список веток
  --help        - справка

Требования

- GCC (MinGW-w64)
- Make
- Windows / Linux
